Amir Goli - Static Personal Website

This is a simple, fast, mobile-friendly HTML/CSS website using a white + mauve theme (#7A4988).

Pages
- Home: index.html
- About: about.html
- Skills & Honors: skills-honors.html
- Publications: publications.html
- Projects (with 6 project pages): projects/index.html + projects/project-1.html ... projects/project-6.html
- Activities: activities.html
- Contact: contact.html

Quick start
1) Open index.html in a browser (double click), or upload the folder to any static host.

Customize
- Replace assets/img/headshot-placeholder.svg with your real photo (keep the filename for an easy swap).
- Replace placeholder images:
  - assets/img/placeholder-wide.svg
  - assets/img/placeholder-square.svg
- Update links in contact.html (GitHub, demos, etc.) as needed.

Theme
- Accent colors live in assets/css/style.css under :root.

Notes
- This is a static template (no build step).
